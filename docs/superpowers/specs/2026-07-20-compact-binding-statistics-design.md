# Port Cleaner 紧凑端口统计设计

## 目标

- 删除独立的“监听端口概览”区域和统计卡片。
- 将总数、TCP、UDP、受限数量和最近刷新时间合并到监听端口表格头部。
- 保留当前筛选结果数量，并在窄窗口中自然换行。

## 方案

统计数据仍基于完整的监听端口集合，筛选结果单独显示为“显示 X / 共 Y”。表格标题左侧继续展示“端口检查 / 监听端口”，右侧使用紧凑的统计项展示 TCP、UDP、受限数量与最近刷新时间。

删除 `SummaryMetrics` 组件及卡片样式，由 `App` 使用 memoized 统计值直接渲染表格头部。统计容器使用可访问名称，便于测试和辅助技术识别。

## 验证

- 组件测试确认独立概览标题和统计卡片不再存在。
- 组件测试确认表格头部包含筛选数量、TCP、UDP、受限数量和刷新时间。
- 运行完整前端检查、构建和 Rust 测试。
