# Port Cleaner macOS 选择框样式修复设计

## 目标

- 消除 macOS WebKit 为协议和权限选择框添加的原生圆角、高光和系统箭头。
- 让选择框与起始端口、结束端口输入框保持一致的高度、边框、背景和文字布局。
- 保留原生下拉菜单、键盘操作和焦点可访问性。

## 方案

协议与权限标签增加选择框专用类和一个仅用于视觉展示的箭头节点。CSS 对 `select` 设置标准与 WebKit 的 `appearance: none`，再使用统一的文本箭头替代系统箭头。

箭头节点不参与鼠标和辅助技术交互；选择框继续使用原生 `select` 元素，不引入自定义下拉菜单或 JavaScript 状态。

## 验证

- React 测试确认两个选择框使用专用容器并包含隐藏于辅助技术的统一箭头。
- 静态样式测试确认关闭标准/WebKit 原生外观并配置箭头定位。
- 运行完整前端测试、构建与 Rust 测试。
